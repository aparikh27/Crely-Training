# Crely-Training

# Project Overview
This repository cnotains the training process for technical foundations in physiological signal processing and clinical ECG interpretation. The goal is to demonstrate readiness for Crely's proprietary multi-modal datasets by mastering open-source tools and public benchmarks. 

# Technical Stack
* Language: Python 3.14
* Datasources: PhysioNet (MIT-BIH Arrhythmia Database, PTB-XL)
* Libraries: `WFDB` (Data I/O), `NeuroKit2` (Signal Processing), `Matplotlib` (Visualization)
